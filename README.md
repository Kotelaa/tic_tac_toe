# Tic Tac Toe

A classic Tic Tac Toe game built in Python. Play against another player in the terminal.

---

## How to play

- Two players take turns: X and O
- Enter the number of the cell you want to mark (1-9)
- First player to get three in a row wins
- If all cells are filled with no winner, it's a draw

---

## Tech Stack

- **Language:** Python
- **Interface:** Terminal / command line

---

## Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/Kotelaa/tic_tac_toe
cd Tic_tac_toe
```

**2. Run the game**
```bash
python main.py
```

No dependencies required – runs with standard Python.
